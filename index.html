ℹ️Connect Firebase to save your data to the cloud — safe, automatic, accessible from any device. Takes about 5 minutes to set up.
How to set up (one-time)
1Go to console.firebase.google.com → Create a project (name it anything)
2In your project → Build → Firestore Database → Create database → Start in test mode → Choose a region
3Go to Project Settings (gear icon) → Your apps → Add app → Web (‹/›) → Register app → Copy the config values below
4Paste each value into the fields below and tap Save
